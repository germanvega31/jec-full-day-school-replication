# jec-full-day-school-replication
# Replication code

This repository contains the replication code for “Full-Day School Programs and Higher Education: Evidence from Peru”.

The raw Young Lives data are not redistributed with this replication package and must be obtained from the original data provider (UK Data Archive).

Users should always verify that they are using the latest version of the code. This code reproduces the main tables and Figure 2 of the paper. The original data should be prepared following the procedure described in the paper. 

This repository will remain publicly accessible.
